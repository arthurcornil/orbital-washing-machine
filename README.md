### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/arthurcornil/orbital-washing-machine.git
   cd ./orbital-washing-machine
   ```
2. Build the project
   ```sh
    make
   ```
4. Launch the game
   ```sh
    ./owm
   ```
